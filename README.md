# Social Media Backend REST API 
# Features:
  1. GET  user profile 
  2. Follow a user 
  3. Upload a post 
  4. Delete a post 
  5. Like a post 
  6. Unlike a liked post
  7. Comment on a post
# Steps to run:
```
  docker compose build
  docker compose up
 ```
  
  Now the server is listening on port 4000 not on 8080, go to http://localhost:4000/
